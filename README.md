# Employee Performance Prediction using Machine Learning

## 📌 Business Problem
INX Future Inc observed a decline in employee performance impacting productivity and business outcomes. The goal was to identify key factors affecting performance and build a predictive model to support HR decision-making.

---

## 🎯 Project Objective
- Identify key drivers influencing employee performance  
- Build a predictive model to classify performance ratings  
- Provide insights to improve employee productivity  

---

## 📊 Dataset
- Records: ~1200 employees  
- Features: 28  
- Target Variable: Performance Rating (2, 3, 4)  
- No missing values  

---

## 🧠 Approach

### Data Preprocessing
- Cleaned and validated dataset  
- Converted data types where necessary  

### Exploratory Data Analysis
- Analyzed department-wise performance trends  
- Identified key influencing variables  

### Feature Selection
- Selected relevant features impacting performance  

---

## 🤖 Models Used
- Logistic Regression  
- Random Forest  
- Gradient Boosting  

---

## 📈 Model Performance
- Best Model: **Gradient Boosting**
- Accuracy: **~93%**
- Strong macro F1 score  

---

## 🔑 Key Insights
- Salary hike percentage strongly influences performance  
- Environment satisfaction impacts productivity  
- Delay in promotions reduces employee performance  

---

## 💡 Business Impact
- Helps HR identify high and low performing employees  
- Supports performance-based salary and promotion decisions  
- Enables targeted employee engagement strategies  

---

## 🛠 Tech Stack
- Python (Pandas, NumPy, Scikit-learn)  
- Machine Learning (Gradient Boosting, Random Forest)  
- Data Visualization (Matplotlib, Seaborn)
